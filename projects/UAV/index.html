UAV Avionics Integration & Flight Testing

This project documents hands-on work integrating and testing a multirotor UAV avionics stack, including flight controllers, ESCs, radio links, GPS, telemetry, and power systems. The focus is on embedded flight control configuration, peripheral integration, and real-world debugging under operational constraints.

Work included configuring flight control firmware, validating PWM motor outputs, integrating UART-based peripherals, and verifying radio failsafe behavior across multiple flight modes. Flight testing was used to evaluate system stability, telemetry integrity, and control responsiveness.

Detailed system architecture, configuration notes, and flight data analysis are currently being consolidated and will be published here. Documentation in progress.
