Simon Game

Microcontroller memory game with keypad, LCD, LEDs, tones, and non-blocking timing.

Firmware UX Timing

Built a fully playable Simon game that balances display and audio updates with responsive input handling. The core loop is non-blocking, with debounced keypad input, LCD rendering, and tone generation sequenced under tight timing constraints.

I implemented pattern generation, keypad handling, audio feedback, and LCD output, aiming for a smooth interaction flow and consistent difficulty pacing across sequences. The main challenge was scheduling peripherals so the UI stayed responsive without starving the game logic.

The final build feels snappy and stable, with clear feedback and a simple path to add difficulty levels or new modes later.
