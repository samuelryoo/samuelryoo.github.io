Simon Game

Interactive visual/audio memorization game, using the AVR RISC-V microcontroller (AT32Mega).

C/C++ Embedded Systems Soldering KiCad

Built a fully playable Simon game that balances display and audio updates with responsive input handling. The core loop is non-blocking, with debounced keypad input, LCD rendering, and tone generation sequenced under tight timing constraints.

Implemented pattern generation, keypad handling, audio feedback, and LCD output for a smooth interaction flow and consistent difficulty pacing. Main challenge: scheduling peripherals so the UI stayed responsive without starving the game logic.

The final build is snappy and stable, with clear feedback and a simple path to add difficulty levels or new modes.
